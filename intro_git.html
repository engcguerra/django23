GIT

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

site oficial
  • sistema de controlo de versões
  • gratuita
  • open source
  • mais usado!
  • muito rápido
  • guarda versões inteiras dos ficheiros
  • não guarda cópias da mesma imagem!
  • sistema distribuído - podemos ter repositório locais e um central

como verificar se temos instalado: git --version

para criar um repositório local: git init

ver como status: git status

comando para adicionar o ficheiro para o staging area/index (área de preparação): git add nomeficheiro

git add nomeficheiro1 nomeficheiro2

comando para adicionar todos os ficheiros da pasta para o staging area/área de preparação: git add .

para gravar um snapshot/ uma versão do nosso código: git commit -m "descricao"

para efetuar o commit é necessário ter definido a informação do autor / configuração do git

git config --global user.name "nome utilizador"

git config --global user.email emailutilizador

para visualizar os vários commit /snapshots efetuados: git log

git log --oneline

git log --oneline --graph

Ramificação do código - criar branch e efetuar o merge

para criar um novo ramo / branch e indicar o seu nome: git branch nomedobranch

para listar os ramos /branch e indicar em qual estamos: git branch

para mudar de ramo / branch: git switch outrobranch

ou git checkout outrobranch (todo: verificar as diferenças): git checkout outrobranch

para eliminar um branch: git branch -d nomedobranch
